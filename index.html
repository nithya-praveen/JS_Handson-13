Q9. Write code to implement timer clock using JS -- display HH:MM:SS -- the time should keep updating every second.

A9.
const date = new Date();
let hours = date.getHours();
let mins = date.getMinutes();
let secs = date.getSeconds();
document.getElementsByClassName('hr').innerHTML=`${hours}`;
document.getElementsByClassName('min').innerHTML=`${mins}`;
document.getElementsByClassName('sec').innerHTML=`${secs}`;
